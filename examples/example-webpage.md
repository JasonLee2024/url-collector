## 内容摘要

Rust 官方异步编程指南（Asynchronous Programming in Rust），系统介绍 Rust 异步编程的核心概念：Future trait 的执行模型、async/await 语法的编译原理、Pin 与 Unpin 的内存安全保证、Waker 与任务调度机制，以及主流运行时（tokio、async-std）的生态概览。适合有一定 Rust 基础、希望深入理解异步运行时原理的开发者。
